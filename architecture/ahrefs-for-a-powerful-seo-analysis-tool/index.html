site-audit - 网站分析

site-audit 会启动爬虫按照爬虫规则来爬取你网站上所有的资源，然后提供整体SEO评分，
可视化图表中的关键数据，将标记出所有可能的SEO问题，并提供有关如何解决它们的建议。

在进行网站分析之前，只需要做一个小小的验证，来证明域名所有权。

下面简单说一下包含的主要功能：

  • 概况（Overview）

概况页面包含几个图表和一个表格。

图表展示了某次抓取的大致情况，

表格则详细指出了抓取过程中遇到的问题，通过跟踪每一类问题，找到涉及当前问题的链接，方便我们有针对性的进行改进。

常见的问题有：title字数太少，页面中没有包含 h1 标签，缺少 TDK（title、description、keywords）等等

  • *抓取日志（Crawl log）

查看某一次网站抓取的情况，可以大致看出在抓取过程中，正常抓取的页面和异常抓取的状态分布。
